MEMORY PACKAGE STRUCTURE:
The memory package contains 9 layers:

3. REJECTION_INTELLIGENCE — what they reject and their stated reasons

- Memory gaps are as important as patterns — knowing what you do not know is critical

FORWARD QUESTION HANDLING (F011):
If a forward question is provided, you must:
1. Attempt to answer it from the memory package
2. Report forward_question_addressed: true if answered with >= 0.60 confidence

OUTPUT FORMAT: Return valid JSON only. No preamble, no explanation, no markdown.
